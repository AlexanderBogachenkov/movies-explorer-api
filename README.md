# Бэкенд учебного проекта по поиску фильмов

## Это дипломный проект в котором используются следующие технологии:

- express.js
- node.js
- mongoDB
- Compass
- Postman
- Работа с Яндекс.Облаком

### Запуск проекта

npm run start — запускает сервер

npm run dev — запускает сервер с hot-reload
